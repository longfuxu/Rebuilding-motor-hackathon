# In-silico 定向进化 —— GREEDY 上位性扫描(背景 = M307W)(2026-07-09)

用户方法学修正:round-2 的"双突变不叠加"只测了 4 个预选双突变(欠采样)。本轮固定round-1 赢家 **M307W 为背景**,在**一大批**被允许的第二位点上构建 M307W+X,直接、更广地检验:**在最好的单点背景上,还有没有第二个突变能叠加(两个突变强于一个)?**

## 设计

- 背景固定 = M307W(不再动 307)。第二位点 X = 生化 TOLERANT {56,99,127,128,222,233} + 未测刚性化界面/核心位点 {100,130,289},每个位点取有刚性化意义的 F/W/Y(核心 L289 另取 Pro)。

- 排除所有 SEVERE/LETHAL(129,58,105,328,330,28,32,106,126,158,234,294)与催化残基。

- 库 = 20 个 M307W+X。其中 **K233F / K233W 无法用 tiled-MSA 折叠**(native 233 位于 segB/linker 拷贝边界,derive_blocks 无法锚定 5 个拷贝——边界伪影,已剔除),故实测 **18 个 M307W+X**(3 个 M307W+N128F/S127F/L289P 复用 round-2 折叠/打分)。

- 每个 M307W+X:tiled-MSA / Boltz-2 折叠 → gated 打分(M1∧M2 handedness-robust∧M5∧soft-mode∧biochem)× power(coord PRS + force-network + M3 grip)。ESM-2 可信度过滤。

## 判定标准(决定性问题)

**叠加(STACKS)= 门槛全过 且(相对 M307W 背景,coord 增益 > 6.35% 噪声地板 或 force 增益 > 1.46% 噪声地板)。** 即:比最好的单点还要好、且超过折叠噪声。

背景 M307W:PRS_NN = 0.002193,conduit = 4.6032(round-1:coord +6.4% / force +3.5% vs WT——这是每个 M307W+X 要超越的线)。

噪声地板(|WT−WTrep|):coord = 6.35%,force = 1.46%。门槛全过的 M307W+X:**16/18**。

## 诚实结论

1. **没有叠加。** 在这个更广的第二位点扫描里,**没有一个门槛全过的 M307W+X 在噪声之上击败 M307W 背景**(coord、force 均未过噪声)。→ round-2 的"单点不叠加"结论现在是**稳健的、更广采样的负结果**:即使在最好的单点背景上,再加一个刚性化点也不能稳健地更强。

2. 机制解读:M307W 已把 Y129↔307 clasp 的网络刚度推到接近最优;在这个已优化的背景上再加约束,要么被噪声淹没,要么(和 round-2 一样)过度刚性化、把协同/力压回去。**"更强"不是靠继续堆刚性化点堆出来的。**

3. **另一条力传导线(S99W)**:M307W+S99W 在此背景上不叠加(dCoord -9.6%,dForce -2.4% vs M307W)。→ N-domain 的 S99W 与 C-domain 的 M307W 合起来并不优于单独的 M307W——两条力传导线不叠加。

4. **门槛稳健性**:16/18 个 M307W+X 全过 M1/M2/M5/soft/biochem——折叠层面这些刚性化都不破坏装配/催化/软模;区分它们的是 proxy,不是门槛。


## 对北极星的意义

"能不能设计出更强的?"→ 更广的 greedy 扫描给出诚实的负结果:**在最好的单点背景上,20 个第二位点里没有一个能稳健叠加**。M307W(以及独立的 S99W)仍是 proxy 层最强的、可单分子验证的"略强"候选;要更进一步需要超出"叠加点突变"的手段(骨架重设计 / RFdiffusion-de-novo connector / 负设计打开软模,见 backlog)。
